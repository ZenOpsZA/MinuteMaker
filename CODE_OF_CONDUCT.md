# Code of Conduct

## Purpose

This project is maintained as a professional, respectful, and constructive collaboration space.

Whether you are contributing code, raising issues, or participating in discussions, you are expected to engage in a way that supports a productive and inclusive environment.

---

## Our Commitment

As the maintainer of this project, I am committed to fostering a community that is:

* respectful
* constructive
* inclusive
* focused on meaningful collaboration

Participation in this project should be free from harassment or hostility.

---

## Expected Behaviour

The following behaviours are expected from all participants:

* Demonstrating respect and professionalism
* Being open to feedback and differing viewpoints
* Providing constructive, solution-oriented input
* Taking responsibility for mistakes and learning from them
* Keeping discussions focused on improving the project

---

## Unacceptable Behaviour

The following behaviours are not acceptable:

* Harassment, intimidation, or discrimination of any kind
* Trolling, personal attacks, or derogatory comments
* Publishing private information without explicit permission
* Disruptive behaviour that undermines productive discussion
* Any conduct that would be considered inappropriate in a professional environment

---

## Scope

This Code of Conduct applies to:

* GitHub issues
* Pull requests
* Discussions
* Any communication directly related to this project

It also applies when representing the project in public spaces.

---

## Enforcement

For this repository, the project maintainer is responsible for enforcing this Code of Conduct.

If unacceptable behaviour occurs, appropriate action will be taken, which may include:

* Requesting changes in behaviour
* Issuing warnings
* Limiting or removing participation
* Blocking or banning contributors if necessary

Enforcement decisions will be made with the goal of maintaining a healthy and productive environment.

---

## Reporting

If you experience or witness unacceptable behaviour, you can report it via:

[heinrichs@zenops.co.za](mailto:heinrichs@zenops.co.za)

Reports will be handled respectfully and with discretion.

---

## Enforcement Approach

Enforcement will generally follow a progressive approach:

* **Correction**: Clarification and request for improved behaviour
* **Warning**: Formal warning regarding conduct
* **Restriction**: Temporary limitation of participation
* **Removal**: Permanent removal from the project

Severity and context will always be considered.

---

## Final Notes

This Code of Conduct exists to support the long-term quality and sustainability of the project.

Respectful collaboration leads to better outcomes for everyone involved.
